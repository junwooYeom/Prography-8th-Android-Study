## Ground Rule
- 스터디 시간 : 2시간
- 요일: 자유롭게 하되, 모든 사람의 동의
- 못나오는 경우 : 스킵 -> 진도는 알아서 빼오세요.
- 당일 정오까지 -> 안되면 그날 스터디룸 비용 지불
- 월요일 고정 -> 어려우면 당일 정오까지 자유롭게 
- 온라인, 오프라인 지각 -> 다음 주 세션 준비
